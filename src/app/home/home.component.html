Let's Craft With Class!

Ready to sprinkle some fresh vibes into your Angular projects? Let's get styling!

Ready to Roll?

All it takes is a few steps to sprinkle some style magic into your Angular app. Let's go!

Install the Package
npm i @dev.spot/css-fusion

Import the Module

In your app.module.ts or any custom module:


import { CssFusionModule } from @dev.spot/css-fusion;
    
@NgModule({
    imports: [CssFusionModule]
})
    
export class YourModuleName {}

Include the Styles

In your styles.scss:

@import "@dev.spot/css-fusion/src/lib/scss/main.scss";
